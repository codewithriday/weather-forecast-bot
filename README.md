# 🌤️ Weather Forecast App

A Python desktop app that shows real-time weather and 5-day forecasts for any city — no API key needed!

## Features
- 🌡️ Current temperature & feels like
- 💧 Humidity & wind speed
- 📅 5-day forecast
- 🔄 Celsius / Fahrenheit toggle
- 🖥️ Clean GUI with Tkinter

## Installation
1. Clone the repo
2. Run: pip install requests
3. Run: python main.py

## Tech Used
- Python + Tkinter
- Open-Meteo API (free, no key needed)
